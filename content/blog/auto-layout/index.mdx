## 基本

![Layer](./basic.png)

まず大事なのが、UIScrollView上に直接ボタンやテキストフィールドを置いてはいけないことです。必ずUIVIew(コンテンツビュー)を起き、その中にボタンやテキストを追加する必要があります。

レイアウトは以下の構成にしてください。

```
UIViewController
  - UIScrollView
    - UIView (contentView)
      - UITextField
      - UIButton
      ...
```
